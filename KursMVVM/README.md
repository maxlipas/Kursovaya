# KursMVVM — Учёт спецодежды

Приложение для автоматизации учёта спецодежды, цехов, работников и фактов выдачи спецодежды.

## Вкладки
- **Цеха** — управление справочником цехов.
- **Спецодежда** — управление видами спецодежды (срок носки, стоимость).
- **Работники** — управление работниками с привязкой к цехам и скидками.
- **Получения** — учёт выдачи спецодежды работникам (дата, роспись).
- **Отчёты** — 5 аналитических отчётов.
- **Графики** — 4 типа интерактивных диаграмм.
- **Импорт/Экспорт** — загрузка и выгрузка данных в CSV, Excel, JSON, XML, PDF.

## Стек технологий
- Avalonia UI 12.0
- .NET 10
- Entity Framework Core 10 + SQLite
- CommunityToolkit.Mvvm
- LiveChartsCore (графики)
- EPPlus (Excel)
- iText7 (PDF)
- CsvHelper (CSV)

## База данных
SQLite с 4 сущностями:
- `Workshop` (цеха)
- `SpecialClothing` (спецодежда)
- `Worker` (работники)
- `Receipt` (получения)

Ограничения:
- срок_носки > 0
- стоимость_единицы > 0
- скидка от 0 до 100
- дата_получения ≤ текущей даты
- ссылочная целостность ON DELETE RESTRICT

Тестовые данные:
- 5 цехов
- 5 видов спецодежды
- 15 работников
- 30 получений
